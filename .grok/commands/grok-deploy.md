# Deploy to Vercel (Grok Build)

Manage deployment and environment configuration for the Cerid AI Marketing site.

**Common flows:**
- `preview` / `deploy` — Build and deploy to Vercel preview or production
- `env` — Manage environment variables (especially for marketing analytics, etc.)
- `logs` — View recent deployment / runtime logs
- `rollback` — Quick rollback guidance

This repo uses Vercel. The command should leverage the Vercel CLI where available and fall back to git-based Vercel deployments.

Grok-native version of deployment workflows.
